# implement 스킬

plan.md의 "다음 작업 (구현 예정)" 항목을 구현합니다.

## 실행 규칙

1. **plan.md 읽기**: "다음 작업 (구현 예정)" 섹션에서 미완료 항목을 확인합니다.
2. **단계별 구현**: 각 항목을 순서대로 구현합니다.
   - 구현 전 관련 파일을 먼저 읽고 구조를 파악합니다.
   - `any`, `unknown` 타입을 절대 사용하지 않습니다.
   - 각 단계 완료 후 `npx tsc --noEmit`으로 타입체크를 실행합니다.
   - 타입 에러가 있으면 즉시 수정합니다.
3. **plan.md 업데이트**: 각 항목 구현 완료 시 plan.md에서 해당 항목을 완료(`[x]`)로 표시합니다.
4. **멈추지 않기**: 모든 항목이 완료될 때까지 계속 진행합니다.
5. **빌드 확인**: 모든 구현 완료 후 `npm run build`로 최종 빌드를 확인합니다.
